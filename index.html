Guide d'utilisation
Pour commencer
1
Charger les donnees Cliquez sur "Donnees CST" pour charger automatiquement les engagements, ou selectionnez un fichier JSON personnalise.
2
Parcourir les engagements Les engagements s'affichent sous forme de cartes. Utilisez les filtres pour trouver rapidement ce que vous cherchez.
3
Evaluer les KPI Cliquez sur une carte pour ouvrir le panneau des KPI et renseigner vos evaluations.
Evaluer les KPI

Pour chaque KPI, vous pouvez selectionner :

  • Confiance : Oui / Non / Probablement - Indique si l'objectif sera atteint
  • Importance : Haute / Moyenne / Faible - Priorite de ce KPI

Vos selections sont sauvegardees automatiquement dans votre navigateur.

Comprendre les couleurs
  • Gris : Pas initie - Aucun KPI evalue
  • Bleu : Partiellement complete - Au moins un KPI evalue
  • Vert : Complete - Tous les KPI evalues
Exporter et Importer

Exporter vers Excel :

  • Cliquez sur le bouton vert "Exporter Excel" en bas de page
  • Un fichier .xlsx sera telecharge avec toutes vos selections

Importer depuis Excel :

  • Utilisez la section "Importer selections" dans la barre d'action
  • Selectionnez un fichier Excel precedemment exporte
  • Vos selections seront restaurees automatiquement

Astuce : Vous pouvez modifier le fichier Excel (colonnes selection_confiance et selection_importance) puis le reimporter pour mettre a jour vos evaluations en masse.

Utiliser les filtres
  • Recherche : Tapez un mot-cle pour filtrer tous les champs
  • Domaine : Filtrez par categorie d'engagement
  • KPI : Affichez uniquement les engagements contenant un KPI specifique
  • Etat : Filtrez par niveau de completion
Sauvegarde des donnees

Important : Vos donnees sont stockees localement dans votre navigateur (localStorage). Si vous effacez les donnees de navigation, vos selections seront perdues. Pensez a exporter regulierement vers Excel pour conserver une sauvegarde.
